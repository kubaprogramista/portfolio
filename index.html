Pokedex API

My second project is all about taking data from an API and make design around it.

From thousands of APIs I chose to go with pokeapi.co that holds every data for each pokemon.

I made a pokedex with over 800 pokemons with possibility to check data for each pokemon.

Hardest thing was to create search function that works in real time, it's far from perfect but I learned a lot while doing it.

Biggest flaw of this project is that pokemons on site sometimes render in random order, which is the result of fetching data two times.
